====Hosten van de WEBGL build====
De WEBGL build van de videoplayer is getest in de Google Chrome webbrowser. De videoplayer werd hierbij gehost via de ‘live preview’-functionaliteit van ‘Brackets’. Dit is in principe hetzelfde als de build hosten op de ‘localhost’ van de computer. 
Bij het hosten van de build op een externe cloud host merkten we op dat de videoplayer de videobestanden niet vond. Dit kwam blijkbaar doordat de toegang van de videoplayer tot de locatie van de videobestanden op de server niet standaard werd toegelaten. Het is dus belangrijk om bij het hosten van de videoplayer de map ‘StreamingAssets’ toegankelijk te maken voor de gebruiker en videoplayer.

=====Gebruik van de videoplayer=====
Het start-menu geeft een overzicht van de drie beschreven animaties. Door één van de drie knoppen aan de klikken start automatisch de video van de geselecteerde animatie. De video kan gepauzeerd worden door te klikken op het scherm of via de spatiebalk. Met de muis kan de gebruiker door de tijdlijn van de video ‘scrubben’. De terug-knop (pijl links bovenaan het scherm) neemt je terug naar het start-menu.
